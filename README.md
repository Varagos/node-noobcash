## Get started

To install yarn do

```
npm install -g yarn
```

```bash
# to use yarn, npm install yarn -g
yarn install # or npm install

# Start bootstrap node
yarn start:dev 0 # or npm run start:dev 0

# On a new terminal, start a client node
yarn start:dev 1
```
